Perfil do usuário
Equipe & Acessos
Níveis de acesso disponíveis:
Administrador — acesso total: gerencia usuários, vê todos os clientes, configura o sistema, acessa relatórios financeiros.
Gerencial — visão completa de todos os clientes e relatórios, sem poder criar/excluir usuários ou alterar senhas de terceiros.
Consultor CS — acesso à própria carteira: vê apenas os clientes onde é responsável, registra reuniões e ações, sem acesso a usuários ou dados financeiros de outras contas.
Regras de alerta
Banco de dados
ℹ️ Como seus dados são salvos: este sistema agora guarda tudo em um banco de dados real (PostgreSQL) no servidor — acessível de qualquer computador, por qualquer usuário autorizado. Exporte o backup regularmente como medida de segurança extra e guarde o arquivo .json em local seguro (Google Drive, Dropbox, etc.).
